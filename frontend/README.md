# Chattara

Chattara is a real-time chat application built using the MERN stack.

## Features
- Secure user authentication
- Real-time messaging using Socket.IO
- Modern UI built with React and Vite
- State management using Redux Toolkit
- Protected routes and session handling

## Tech Stack
- React + Vite
- Redux Toolkit
- Node.js & Express
- MongoDB
- Socket.IO

## Setup Instructions

```bash
npm install
npm run dev
